# Game Design Document
## Core Concept
Just a pacman clone with a crab/bird theme.
This project main purpose is teaching me rust and bevy.

## Main Features
 - 2D top down graphics
 - The player (a crab) will run around a maze while avoiding the enemies (birds).

## Target Platform / Audience
 - The game will be made first for PC linux, maybe for website like itch.io
 - No other platforms are planed at this time.

## Timeline / Development Milestones
As of writing is there no dates or milestones, The game is currently only made by me as a pet project.
